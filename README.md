# Decide Storytelling App
This project is a graduation project for Koç University's Computer Engineering program, made for COMP 491 course.

## Project Structure

| Directory | Description | Technology |
|---|---|---|
| `reader-app/` | Mobile app for **readers** — browse and read interactive stories | React Native + TypeScript |
| `author-web/` | Web app for **authors** — create and manage stories, episodes, and decisions | Vite + React + TypeScript |
| `backend/` | API server and database | Python FastAPI + Prisma + PostgreSQL |

## Getting Started

### Reader App (Mobile)
```bash
cd reader-app
npm install
npx react-native start
```

### Author Web
```bash
cd author-web
npm install
npm run dev
```

### Backend
```bash
cd backend
pip install -r requirements.txt
python main.py
```
